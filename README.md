# Contribution Snake

GitHub Actions workflow that generates a contribution snake animation GIF for the profile README.

## Stack

- **Platform:** GitHub Actions\n- **Output:** SVG animation\n- **Purpose:** Profile decoration

## Quick Start

 `bash
n/a — runs on GitHub Actions schedule
 ` 

## How It Works\n\nA scheduled GitHub Action generates a snake animation that crawls across the contribution graph, then commits the result to the profile repo.
